# Регистрация и аутентификация пользователей с загрузкой изображений

## Описание проекта

Этот проект представляет собой веб-приложение на Python с использованием Flask для регистрации и аутентификации пользователей. Пользователи могут зарегистрироваться, войти в систему, обновить свой профиль и загрузить изображение профиля, которое хранится в базе данных в формате BLOB.

## Структура проекта

- `run.py`: Основной файл для запуска приложения. Он создает все таблицы в базе данных при первом запуске.

- `app/__init__.py`: Содержит функцию `create_app`, которая инициализирует приложение Flask и подключает необходимые расширения, такие как SQLAlchemy для работы с базой данных и LoginManager для управления пользователями.

- `app/forms.py`: Содержит формы для регистрации, входа и обновления профиля пользователя. Используются Flask-WTF и валидация данных.

- `app/models.py`: Определяет модель `User` для работы с пользователями в базе данных. Включает функции для установки и проверки хешированных паролей.

- `app/routes.py`: (Не указан в предоставленных файлах, но предположительно содержит маршруты для обработки запросов пользователей. Это файл, где определяются обработчики для регистрации, входа, обновления профиля и другие функции.)

- `templates/`: Папка, содержащая HTML-шаблоны для представления страниц веб-приложения (например, `login.html`, `register.html`, `profile.html` и т.д.).

- `bd_view.py`: Сервисный файл для вывода на терминал содержимого БД.

- `static/`: Папка, содержащая изображения пользователей (для возможного использования при проверки загрузки в БД).

## Установка и запуск

1. **Клонирование репозитория**

   ```bash
   git clone <репозиторий>
   cd <директория проекта>
   ```

2. **Создание виртуального окружения**

   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate  # Windows
   ```

3. **Установка зависимостей**

   Убедитесь, что у вас установлен pip, затем выполните:

   ```bash
   pip install -r requirements.txt
   ```

4. **Настройка базы данных**

   Приложение использует SQLite. Таблицы создаются автоматически при первом запуске.

5. **Запуск приложения**

   ```bash
   python run.py
   ```

   Перейдите по адресу `http://127.0.0.1:5000` в вашем браузере.

## Основные функции

- **Регистрация**: Пользователи могут зарегистрироваться, указав имя пользователя, email, пароль и загрузив изображение профиля.
- **Аутентификация**: Вход в систему с использованием email и пароля.
- **Обновление профиля**: Пользователи могут обновлять свои данные и изображение профиля.
- **Загрузка изображений**: Изображения профиля хранятся в базе данных в формате BLOB.

## Важные замечания

- Убедитесь, что `SECRET_KEY` в `app/__init__.py` защищен и не должен быть загружен в публичные репозитории.
- Приложение использует `Flask-Login` для управления аутентификацией пользователей.
- Изображения загружаются и хранятся в базе данных в поле `LargeBinary`.
